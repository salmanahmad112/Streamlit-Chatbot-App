# Streamlit Chatbot App

This project is a simple chatbot web application developed using Streamlit, LangChain, and Groq. It provides a basic interface where the user can enter a question and receive a response generated by the model. The main purpose of this project is to show how a chatbot can be built in a simple and beginner-friendly way by combining a web interface, prompt handling, and model integration in one application.

The application is developed in Python. Streamlit is used to create the frontend interface, LangChain is used to manage the prompt and output flow, and Groq is used as the language model provider for generating responses. The project also uses python-dotenv to load the API key from an environment file.

To run this project, the required libraries should first be installed using the requirements file. After installing the dependencies, the user needs to create a `.env` file in the project folder and add the Groq API key. Once this is done, the application can be started with the Streamlit command and opened in the browser for use.

This project is helpful for beginners who want to understand the basic structure of a chatbot application using Python tools. It can also be improved further by adding chat history, better prompt design, and more advanced user interface features in the future.
